# 黑板实验室 · Blackboard Lab

两套课程的交互式可视化配套教程，顶部页签切换：

- **MIT 18.06 线性代数**（Gilbert Strang）——把「矩阵作为变换 / 列的线性组合」的几何图像做成可以拖、可以玩的东西；
- **Zero to Hero 神经网络**（Andrej Karpathy）——从导数、概率这些零基础概念讲起（文字教程 + 嵌入式小交互），再到在浏览器里**真实训练**的模型：反向传播逐步动画、梯度下降实验台、Bigram 语言模型现场训练与采样。

核心理念一致：不做计算器，做直觉。

## 运行

```bash
npm install
npm run dev      # 开发服务器，浏览器打开提示的地址
npm run build    # 类型检查 + 打包到 dist/
```

纯前端，无任何运行时依赖（Vite + TypeScript + Canvas）。

## MIT 18.06 模块与课程对照

界面按课程三个单元组织（“黑板 + 讲义纸”设计：画布是粉笔板书，控制面板是讲义纸）。

**单元一 · Ax = b**

| 模块 | Lecture | 内容 |
| --- | --- | --- |
| 行图像 vs 列图像 | L1 | 同一个 Ax=b 的两种看法并排联动：左图直线交点，右图列组合 x·c₁ + y·c₂ 逼近 b |
| 消元与 A = LU | L2–4 | 3×3 增广矩阵逐步消元：主元/被消位置高亮、每步的初等矩阵 E、行交换（PA = LU）、L 与 U 汇总；旁边的 3D 行图像里三张平面随每步变化而交点不动 |
| 线性变换 | L3·30 | 网格从单位阵连续形变到 A；î、ĵ 落到 A 的两列；单位正方形的像 = det；可叠加特征方向 |
| 四个基本子空间 | L6–10 | 秩 1 矩阵把输入空间劈成 C(Aᵀ) ⟂ N(A)；拖动 x 看分解 x = x_row + x_null 与 Ax 落在 C(A) |

**单元二 · 正交与行列式**

| 模块 | Lecture | 内容 |
| --- | --- | --- |
| 投影与最小二乘 | L15–16 | 左图拖 a、b 看投影 p 与垂直误差 e；右图拖数据点看正规方程拟合直线与残差 |
| Gram-Schmidt 与 QR | L17 | 分步演示 q₁ = a/‖a‖、B = b − (q₁ᵀb)q₁、q₂ = B/‖B‖，含 R 矩阵与共线退化情形 |
| 行列式与面积 | L18–20 | 拖两列向量看有向面积；按钮验证交换变号、按列线性、消元不变 |

**单元三 · 特征值**

| 模块 | Lecture | 内容 |
| --- | --- | --- |
| 特征向量 | L21–22 | 拖 x 找 Ax ∥ x 的方向；幂迭代动画收敛到主特征方向 |
| 差分方程与 Aᵏ | L22 | u_k = Aᵏu₀ 轨迹；u₀ 按特征方向分解，λᵏ 的增长/衰减一目了然（斐波那契、马尔可夫预设） |
| 微分方程与相图 | L23 | du/dt = Au 的向量场 + RK4 解轨迹动画；按特征值分类鞍点/结点/螺旋/中心 |
| 马尔可夫矩阵与稳态 | L24 | 概率单纯形上拖动 u₀，看分布被吸向 λ=1 的稳态 π；收敛速度 ≈ \|λ₂\| |
| 对称矩阵与正定性 | L25–28 | 二次型等高线（椭圆/双曲线）+ 特征方向主轴；主元、特征值、顺序主子式三组判据 |
| 奇异值分解 SVD | L29 | 单位圆 → 椭圆的三段动画 Vᵀ→Σ→U，含 σ 值与 UΣVᵀ 重构验证 |
| 基变换与图像压缩 | L31 | 8×8 画板上作画，DCT 基下看系数分布，保留前 k 个系数重构——体会“换基让信息集中” |

## 操作提示

- 所有画布支持**滚轮缩放**；画布上能拖的东西（向量尖端、数据点、初始点）直接拖。
- 矩阵输入框可以直接改数字，每个模块都有预设下拉菜单，选中即演示。
